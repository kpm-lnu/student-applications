# Crack Reconstruction Using Nonlinear Integral Equations

Цей проєкт присвячений розв'язанню **оберненої задачі** — реконструкції форми та просторового розташування внутрішньої тріщини ($\Gamma_1$) у двовимірному тілі з відомою зовнішньою границею ($\Gamma_2$).

В основі алгоритму лежить метод нелінійних інтегральних рівнянь та ітераційна схема із застосуванням **регуляризації Філіпса-Тіхонова**..

Проєкт складається з двох основних частин:

1. **Пряма задача** (`direct_problem.ipynb`) — перевірка, що ядро алгоритму для розв'язування обрененої задачі працює коректно; для генерування вхідних даних для оберненої задачі.
2. **Обернена задача** (`crack_reconstruction_exact_data.ipynb` та `crack_reconstruction_noisy_data.ipynb`) — програмна реалізацію основного алгоритму для реконструкції тріщини для точних вхідних даних та даних з шумом.

## 3. Математичне моделювання (Python / Jupyter Notebook)

### Опис констант та параметрів алгоритму

Основні параметри:

- `N = 16` — кількість точок дискретизації для кривої.
- `M = 4` — параметр для методу колокації.
- `ALPHA, BETA, GAMMA = 1e-4` — параметри регуляризації Філіпса-Тіхонова для стабілізації розв'язку.
- `KSI = 0.1` — параметр затухання.
- `EPS = 1e-3` — критерій зупинки ітераційного процесу (точність реконструкції).
