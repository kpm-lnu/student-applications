# FEM Elasticity Coursework

Програмна реалізація для курсової роботи:

**«Застосування двовимірної моделі до задачі теорії пружності із використанням методу скінченних елементів»**

## Що реалізовано

Програма розв'язує три двовимірні задачі лінійної теорії пружності методом скінченних елементів:

1. Пластина з круглим отвором під розтягом.
2. Консольна балка під дією вертикальної сили.
3. Товстостінне кільце під внутрішнім тиском.

Використовуються лінійні трикутні скінченні елементи з двома ступенями вільності у кожному вузлі: `u_x`, `u_y`.

## Встановлення

```bash
pip install -r requirements.txt
```

## Запуск

Запустити всі приклади:

```bash
python main.py --example all
```

Запустити один приклад:

```bash
python main.py --example plate
python main.py --example beam
python main.py --example ring
```

Запустити аналіз впливу густини сітки:

```bash
python main.py --example convergence
```

## Результати

Результати зберігаються в папці `results/`:

- текстовий звіт `zvit.txt`;
- сітка;
- деформована сітка;
- поле переміщень;
- поле напружень за Мізесом;
- таблиця порівняння `summary.csv`.
