# МСЕ-солвер двовимірної задачі адвекції-дифузії

Програмна реалізація до курсової роботи: метод скінченних елементів
на P1 трикутних елементах із стабілізацією SUPG для режиму великих
чисел Пекле.

## Структура

```
src/                  модулі солвера: сітка, базис, квадратури,
                      збирання матриць, граничні умови, час, SUPG
main.py               запуск шести числових експериментів
convergence_study.py  дослідження збіжності, ~15 хв
verify_vs_skfem.py    звірка з бібліотекою scikit-fem
make_gifs.py          GIF-анімації прикладів
results/              сюди зберігаються рисунки та анімації
```

## Як запустити

Потрібен Python 3.10+. Встановити залежності:

```bash
pip install numpy scipy matplotlib pillow scikit-fem
```

Запустити шість прикладів (швидко, ~1 хв):

```bash
python3 main.py
```

Запустити дослідження збіжності (повільно, ~15 хв на 8 ядрах):

```bash
python3 convergence_study.py
```

Звірити власну реалізацію з бібліотекою `scikit-fem` на задачі чистої
дифузії:

```bash
python3 verify_vs_skfem.py
```

Згенерувати GIF-анімації для прикладів~1–3:

```bash
python3 make_gifs.py
```

Усі результати з'являться у папці `results/`.
